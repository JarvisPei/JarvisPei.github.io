Software

This page collects public code repositories and software artifacts from my recent work.

LLM Agents and Long-Context Learning

SCOPE

Self-evolving Context Optimization via Prompt Evolution.

  • Problem: LLM agents receive large, dynamic execution contexts but often fail to manage useful signals from traces, errors, and tool outputs.

  • Method: Treats context management as online prompt optimization, evolving agent prompts from execution traces.

  • Links: Paper, Code.

  • Keywords: LLM agents; prompt evolution; context optimization.

MemDLM

Memory-Enhanced DLM Training.

  • Problem: Diffusion language models need stronger training signals for long-context and denoising dynamics.

  • Method: Introduces memory-enhanced training with a parametric memory channel.

  • Links: Paper, Code.

  • Keywords: diffusion language models; parametric memory; long-context training.

Efficient LLMs and Model Compression

CMoE

Analytical FFN-to-MoE Restructuring via Activation Pattern Analysis.

  • Problem: Dense LLMs are expensive at inference time, while training MoE models from scratch is costly.

  • Method: Converts dense FFNs into sparse MoE layers through analytical post-training restructuring based on activation patterns.

  • Links: Paper, Code.

  • Keywords: FFN-to-MoE; analytical restructuring; LLM acceleration.

PreMoE

Proactive Inference for Efficient Mixture-of-Experts.

  • Problem: MoE models are usually deployed with full expert capacity even when a deployment scenario may only need a subset.

  • Method: Compiles sparse MoE variants before serving through expert pruning and deployment-specific sparsity.

  • Links: Paper, Code.

  • Keywords: MoE expert pruning; proactive inference; efficient deployment.

FuseGPT

From Pruning to Grafting: Dynamic Knowledge Redistribution via Learnable Layer Fusion.

  • Problem: Structured pruning removes whole transformer blocks and can discard useful knowledge.

  • Method: Reframes pruning as learnable knowledge grafting, redistributing knowledge from pruned blocks into surviving layers.

  • Links: Paper, Code.

  • Keywords: structured pruning; layer fusion; knowledge grafting.

LLM Evaluation

Behavioral Fingerprinting

Behavioral Fingerprinting of Large Language Models.

  • Problem: Accuracy-only benchmarks miss cognitive and alignment-style differences between LLMs.

  • Method: Profiles models through behavioral dimensions such as reasoning, robustness, sycophancy, and interaction style.

  • Links: Paper, Code.

  • Keywords: LLM evaluation; behavioral fingerprinting; alignment robustness.
